Abstract

Algorithms

AdaBoost Classifier: AdaBoost (Adaptive Boosting) is a popular ensemble learning method used for classification tasks. It combines multiple weak classifiers to create a strong classifier. Each weak classifier is trained sequentially, with the emphasis on misclassified instances in the subsequent classifiers. The final prediction is determined by a weighted majority vote of the weak classifiers. Algorithm Workflow:
Data Preprocessing: Categorical attributes are encoded using LabelEncoder to convert them into numerical values. The "day" column is dropped from the dataset as it doesn't contribute to the prediction.
Feature Selection: Two features, namely humidity and wind, are selected for predicting whether to play tennis.
Model Training: The dataset is split into training and testing sets using train_test_split function. AdaBoostClassifier is trained on the training data.
Model Evaluation: The accuracy of the trained model is evaluated on the test data using accuracy_score metric.
Prediction: The trained model is used to predict whether to play tennis based on new input data.
